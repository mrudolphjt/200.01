# 200.01 eSeries | Login & Logout

GitHub Pages-ready interactive course for Rippling embed testing.

## Files
- `index.html` — course page for GitHub Pages
- `assets/` — screenshot assets used by the course
- `preview-standalone.html` — self-contained local preview with embedded images
- `.nojekyll` — helps GitHub Pages serve static assets
- `rippling-embed.html` — basic iframe snippet

## GitHub Pages
Upload all files and folders to the root of a GitHub repository, then enable Pages from the `main` branch and `/root`.

## Rippling
Use the published GitHub Pages URL in Rippling's Embed block.
